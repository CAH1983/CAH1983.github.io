Hello World

My name is Caroline, I am a junior full stack web developer. With a strong work experience in major groups across different industries, I discovered the fascinating world of IT thanks to my last occupation as a technical recruiter.

As a firm believer that one can develop new skills through hard work and dedication, I started teaching myself HTML5 and CSS3. This learning experience contributed to and confirmed my genuine interest in websites and digital trends.

PROJECTS

Clubbing Messe

Individual project | Full stack web app.
This final project offers an innovative platform to clubbing communities to showcase upcoming electronic music events and share pictures across the globe.

Users can post, comment or create their own events, or attend existing events posted by others. They can also view other members’ profiles and post pictures related to past events.

The app was coded using Flask framework and Python for the back-end side, as well as HTML, SCSS, JavaScript and React.js for the front-end.

Challenges: We had never used Python | Flask framework, nor PostGreSQL databases beforehand. Trying to retrieve and send data to the front-end was complex.

Wins:

  • UI / UX Design
  • Management and update of SQL database
  • Felt comfortable and enjoyed Python in terms of simplicity of syntax and debugging.
  • Grasp of Insomnia for testing the retrieve of UI data

Treasured

Teamwork project | Full stack web app.
With another developer, we decided to create a CtoC fashion boutique app to resell used clothes.

Sellers can post clothing items including name, description, and price for each. They can, of course, not only create an account and edit their user profile, but also leave comments on items posted by other users.

The key feature is an internal messaging system allowing buyers to reach out to sellers and ask questions.

On the technology side, we used primarily React.js and Node.js, and coded most of the project in pair programming. I focused on the creation of the React.js components, took care of the app’s design, and managed the project to fruition using GitHub and Trello.

Challenges:

  • Retrieve data, set up and styling React
  • Debugging

Wins:

  • Practice of React
  • Excellent team work and coordination
  • Debugging

InstaKlone

Individual project | Full stack web app.
This 2nd project is a streamlined, instagram-like application allowing users to post and comment photos.

The front-end was built using HTML5, CSS, Bulma, EJS and JavaScript. The back-end: with Node.js and the data managed via MongoDB.

Challenges:

  • Set up of Node JS
  • Connecting routers and controllers
  • Manage a project at a bigger scale.

Wins:

  • Better understanding of Node, the MVC pattern and how the different views display.
  • Solid grasp of Bulma framework.
  • Enhance problem solving skills
  • Implementation of a responsive navbar ("burger menu").

The Coding Turtle

The main technologies used are HTML5 + audio, CSS, jQuery.

Challenges:

  • Responsive design

Wins:

  • Use of SetInterval / SetTimeout
  • Use of animation, transitions
